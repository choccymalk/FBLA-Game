How to run:
1. Extract `FBLA-Game.zip` to your desktop, make sure the path is `C:/Users/yourname/desktop/FBLA-Game/`
2. Make sure you have [Java](https://download.oracle.com/java/23/archive/jdk-23.0.2_windows-x64_bin.msi) installed.
3. In the repository directory, run `./gradlew.bat run` in a terminal.
